A frontend project created for a course at University. It's an indie rock band's website, built with HTML, CSS, JavaScript, and Bootstrap. The site features a alternative design, showcasing the band's biography, music, news, and multimedia content. Using Bootstrap to build the main components, while custom JavaScript adds interactive elements, creating some fun interaction.
